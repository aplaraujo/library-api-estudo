Comando para interromper o contêiner: docker stop id-do-container ou nome-do-container

Comando para excluir o contêiner: docker container rm nome-do-container

Comando para criar rede entre contêineres: docker network create nome-da-rede

docker run --name librarydb -p 5432:5432 -e POSTGRES_PASSWORD=postgres -e POSTGRES_USER=postgres -e POSTGRES_DB=library -d --network nome-da-rede postgres:18-alpine
